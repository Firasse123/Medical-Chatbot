# Medical-Chatbot

## Steps to run the project
```bash
conda create -n mchatbot python=3.8 -y
```


```bash
conda activate mchatbot
```

```bash
pip install -r requirements.txt
```

```bash
# run the following command
python store_index.py
```

```bash
# Finally run the following command
python app.py
```
